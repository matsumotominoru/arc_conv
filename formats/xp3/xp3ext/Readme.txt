		吉里吉里/KAG データファイル展開ツール [xp3ext] 2010年

なんですか？
	吉里吉里/KAGの .xp3 ファイルを展開するツールです。
	同様のツールが巷にあるのですが、吉里吉里 2.30 以降でちょっとだけ
	データ形式が変わっていて、それに対応したものが無かったので
	作りました。

超注意？
	「このツールはcygwin上でなければ動きません」。
	だって我輩Windows上の開発環境持ってないんだもの…。
	Win32アプリとして移植は非常に容易だと思いますが、そういう理由なので
	移植の予定はありません。
	あとsetlocale(LC_ALL, "ja_JP.SJIS")とかハードコードしちゃったので
	cygwinでもSJIS環境じゃないとうまいこと動かないです多分。カッコ
	よさに惚れろ！

つかいかた？
	引数に.xp3ファイルを渡すと、何も言わずにカレントディレクトリに
	全部展開します。超男らしいです。
		# xp3ext.exe <.xp3ファイル>
	普通はこんなカンジで使うでしょう。
		# mkdir a; cd a
		# xp3ext.exe ../data.xp3
	え、コマンドラインオプション？そんな軟弱なものは存在せぬわ！
